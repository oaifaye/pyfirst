# Spectral Clustering

A python implementation of Spectral Clustering, tested on twocircle dataset of sklearn.

## Usage

```bash
sudo chmod u+x run.sh
./run.sh
```

## Result

Spectral clustering on the left, and ordinary Kmeans on the right.
